## Como funciona

Com a diretiva `@include`, usar o mixin de espaçamento:

- Onde temos os mixins `margin` e `padding`
- Onde os parâmetros de direção: `x`, `y`, `l`, `r`, `t` or `b`
- Onde os parâmetros de tamanho: `4xs`, `3xs`, `2xs`, `xs`, `sm`, `md`, `lg`, `1xl` ou ,`2xl`

```scss
@include margin(direcao, tamanho);
```

```scss
@include padding(direcao, tamanho);
```

## Uso

### Importar o SASS na página

```scss
@import 'assets/spacing.scss';
```

### Na classe, usar a diretiva include para inserir o código mixin

<b>margin</b>
```scss
.same-class {
  @include margin(x, sm);
}
```

<b>padding</b>
```scss
.same-class {
  @include padding(x, sm);
}
```
