# Border Radius

## Default corner radius Radius: 16px

`  .default-border-radius {
    border-radius: 16px;
  }`

### Para utilização em componentes:

1. Panels
2. Images
3. Cards
4. Cells
5. Forms
6. Modals



## Smalll corner radius Radius-8px / Radius-4px:

`.small-border-radius-8 {
    border-radius: 8px;
}`

`.small-border-radius-4 {
    border-radius: 4px;
}`

### Para utilização em componentes gerais.


## Full corner radius:

`.full-border-radius {
    border-radius: 100%;
}`

### Para utilização em componentes:

1. Buttons
2. Chips
3. Search input
4. Icons
