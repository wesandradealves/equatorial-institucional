# Border Width
Guia de estilos utilizados para Border width para construção dos compoentes e das páginas.

```
$border-width-variants:(
  none:(
    border:none
  ),
  border-width-1:(
    border: 1px solid black
  ),
  border-width-2:(
    border: 2px solid black
  ),
  border-width-3:(
    border: 3px solid black
  )
);

@mixin border-width($variant) {
  @each $name, $style in map-get($border-width-variants,$variant) {
    #{$name}:$style;
  }
}
```
Foi adicionado algumas guia de estilos para padronizar as bordas que seram utilizadas no projeto.
Para se estar utilizando basta estar importando o arquivo ``@import 'assets/border-radius.scss`` no compoente que você
queira utilizar. Depois na classe escolhida, inclua a função que ira trazer a variação escolhida.

```
@import 'assets/border-width.scss';

.estyle-button {
  @include border-width(border-width-2);
}
```
